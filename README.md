# chef-wrapper-consul-cookbook

A wrapper cookbook to install Consul.

## Supported Platforms

Ubuntu 14.04

## Usage

### chef-wrapper-consul::default

Include `chef-wrapper-consul` in your node's `run_list`:

```json
{
  "run_list": [
    "recipe[chef-wrapper-consul::default]"
  ]
}
```
